# Playsic
PLAYSIC, a Python Flask-based Spotify playlist generator, crafts personalized playlists by user input—genre, language, and release years. Seamlessly integrated with Spotify's API, it curates custom playlists directly onto the user's Spotify account, revolutionizing music discovery for tailored listening experiences.
PLAYSIC stands as an innovative Spotify playlist generator driven by Python Flask, designed to empower users in tailoring their musical journeys. Through an intuitive interface, users input preferences—genre, language, and release years—guiding PLAYSIC's algorithm. Seamlessly integrating with Spotify's API, it navigates the vast musical landscape to craft custom playlists. These personalized collections are then directly added to the user's Spotify account, fostering an unparalleled music discovery experience tailored uniquely to individual tastes.
